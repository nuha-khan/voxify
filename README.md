# 🗳️ Voxify – AI Election Assistant

## 🌟 Overview

Voxify is an AI-powered election assistant designed to help users understand election concepts in a clear, structured, and interactive way.

It focuses specifically on election-related queries and delivers responses in three modes:

* ⚡ Simple
* 📘 Detailed
* 🪜 Step-by-step

---

## 🚀 Features

* 🧠 Smart intent detection (process, eligibility, EVM, etc.)
* 🎯 Focused responses only on election topics
* 🔄 Multiple response modes (Simple / Detailed / Steps)
* 💬 Chat-based interactive UI
* ✨ Smooth animations and clean modern design
* 🚫 Handles irrelevant queries with a fallback response

---

## 🛠️ Tech Stack

* 🎨 Frontend: Angular
* ⚙️ Backend: Node.js + Express
* 🎯 Styling: Custom CSS (glassmorphism + gradients)

---

## ⚙️ How It Works

1. 👤 User enters a query
2. 🧠 Backend detects intent using keyword-based logic
3. 🎛️ Selected mode determines response format
4. 💡 Response is generated and sent to frontend
5. 💬 UI displays response with animations

---

## 💡 Example Queries

* "What is the election process?"
* "Who can vote in India?"
* "What is an EVM machine?"
* "Types of elections in India"

---

## 📌 Assumptions
